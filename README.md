## 📝 Descripción del Proyecto
Este programa automatiza de manera interactiva el control de un menú de restaurante mediante el uso de estructuras de datos (matrices) y modularidad. El sistema evalúa dinámicamente las categorías y los precios base de los productos para aplicar de forma automatizada un 15% de descuento a los "Platos Fuertes" cuyo costo supere el umbral de los $25,000.
